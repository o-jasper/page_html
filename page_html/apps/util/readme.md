Some utility functions, for the command userscript.

#### Docs

* `man.lua` man pages.(uses `/usr/bin/man --html` output)

* `doc.lua` documentation pages from files in there.

* `pydoc.lua`, uses `/usr/bin/pydoc`(not tested much)
